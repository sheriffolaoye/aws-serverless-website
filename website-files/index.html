This website's architecture

This is a serverless website put together using some of solutions provided by Amazon Web Services (AWS).

  • AWS S3: AWS Simple Storage Service is used to store and host the website files.
  • AWS RDS: A MySQL database is used to store information about currently hosted repositories in my Github account. This is to avoid making request to the Github API everytime a user visits the website.
  • AWS Lambda: Two Lambda functions are used. The first function uses Github API to update the repositories in the database. The second function acts as a rest API, it serves JSON content containing details of my github repositories currently stored in the database.
  • AWS ApiGateway: AWS API Gateway is used as a trigger for the lambda function that serves as the rest API.
  • AWS CloudWatch: A Cloudwatch rule is used as a trigger for the lambda function that updates the database. A cron expression was used to schedule updating the database.

View source code for the website on Github. This project was inspired by Devon Crawford's website and the A Cloud Guru platform.
