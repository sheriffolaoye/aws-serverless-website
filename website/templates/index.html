This website's architecture

This website uses:

  • EC2: This is a website running on an EC2 t2.nano linux server.
  • Nginx: As a reverse proxy server. Requests to the website's rest API(rest.sheriffolaoye.com) are proxied to gunicorn, a Python WSGI HTTP server, running as a systemctl service. Requests to sheriffolaoye.com are handled normally by nginx.
  • Gunicorn: WSGI HTTP server for running the rest api Flask web app.
  • Flask: The rest api was built using Flask.
  • MySQL: Database for storing repository information.
  • Cron Job: For running a python script that regularly updates the database with the latest repository information.
  • Route 53: For the Domain Name Service(DNS).

View source code for the website on Github. This project was inspired by Devon Crawford's website.
